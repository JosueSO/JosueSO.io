Texto en HTML

Fecha: 30 de Enero de 2017

Encabezados

Para resaltar un texto que indique algun título se suelen utilizar una colección de etiquetas que va de h1 a h6, donde h1 es de mayor importancia y h6 de menor importancia:

<h1> h1 </h1>

<h2> h2 </h2>

<h3> h3 </h3>

<h4> h4 </h4>

<h5> h5 </h5>
<h6> h6 </h6>

Párrafos

Para separar el texto en párrafos se utiliza la etiqueta p:

<p> Contenido del párrafo </p>

Salto de línea

Cuando escribimos un texto, a veces es necesario dar un salto de línea, pero aunque hagamos un salto de linea al escribir un documento HTML, este no será detectado si no se utiliza la etiqueta br. Came mencionar que esta etiqueta no tiene etiqueta de cierre:

<br> Salto de linea

Espacios en blanco

Al escribir un texto largo, el navegador nos hace un cambio de línea automático para que todo el texto pueda observarse en la pantalla. Esto ocurre de igual manera cuando cambiamos el tamaño de la ventana del navegador, se hace un autoajuste de linas de texto, y esto puede ocacionar que las palabras se separen. Si no queremos que dos palabras se separen utilizamos la etiqueta &nbsp; (Non-breaking space) que fuerza el espacio entre palabras:

Palabras &nbsp; juntas

Negritas

Para remarcar un texto en negritas se utiliza la etiqueta b:

<b> Negritas </b>

Cursiva

Para hacer un texto en cursivas se utiliza la etiqueta i:

<i> Cursiva </i>

Resaltado

Para resaltar un texto se utiliza la etiqueta strong:

<strong> Resaltado </strong>

Énfasis

Para hacer un énfasis un texto se utiliza la etiqueta em:

<em> Énfasis </em>

Caracteres Especiales

Como te habras dado cuenta en un documento HTML utilizamos mucho algunos caracteres así que no podemos ponerlos en texto como tal cuando necesitamos mostrarlos en el navegador, para ellos utilizamos un tipo de codificación. Aquí algunos ejemplos.s

&#60; → <
&#62; → >
&#38; → &
&#35; → #
