jQuery

Fecha: 7 de Marzo de 2017

"Write less, do more"

jQuery es una biblioteca multiplataforma de JavaScript, creada inicialmente por John Resig en 2006, que permite simplificar la manera de interactuar con los documentos HTML, manipular el árbol DOM, manejar eventos, desarrollar animaciones y agregar interacción con la técnica AJAX a páginas web.
Simplifica:

  • La interacción con HTML
  • La manipulación del DOM
  • La manipulación de CSS
  • Manejo de eventos
  • Animaciones
  • Interacciones Ajax
  • Garantiza la compatibilidad entre navegadores

Todo se realiza a través de una función llama jQuery(). Recibe como parámetro una expresión CSS y regresa un objeto jQuery

var objeto = jQuery('selector_css')

Filtros básicos de selectores

  • :first → primer elemento
  • :last → último elemento
  • :even → posiciones par
  • :odd → posiciones impar
  • :eq(index) → igual al índice
  • :gt(index) → más allá de la posicion
  • :lt(index) → antes de una posición
  • :not(selector) → sin el selector
