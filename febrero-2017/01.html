Enlaces e Imágenes en HTML

Fecha: 1 de Febrero de 2017

Enlaces

Probablemente la característica que más ha influido, junto con las imágenes, en el espectacular desarrollo de la Web hayan sido los enlaces (links). Un enlace aparece generalmente como un texto azul subrayado y cuando situamos el cursor sobre él se transforma en una mano con el dedo índice extendido. Si pulsamos sobre el enlace saltamos a otra parte del documento, a otro documento situado en cualquier lugar, o incluso se abre el programa de correo para enviar un mensaje a la dirección indicada.

Un enlace lo definimos por la siguiente etiqueta:

<a href="url_del_enlace" target="" > Texto del enlace </a>

El atributo targer sirve para indicar el comportamiento del enlace:

  • _blank es para indicar que el enlace se abrirá en otra pestaña o ventana
  • _self es para indicar que el enlace se abrirá en la misma pestaña

Ejemplo:

<a href="http://www.google.com" target="_blank" > Google </a> → Google

Imágenes

Al momento de diseñar una página muchas veces es necesario adjuntar una imagen, ya sea para usarla como logotipo de la página, como parte del diseño, o para ejemplificar algo. Para añadir una imagen utilizamos la siguiente etiqueta:

<img src="url/nombre_imagen" height="alto" width="ancho" alt"texto aleternativo ">

Ejemplo:

<img src="https://www.w3.org/html/logo/downloads/HTML5_Logo_512.png" alt="Logo HTML5" width="100px" > →
