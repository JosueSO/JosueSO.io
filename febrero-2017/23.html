Arreglos JavaSript

Fecha: 23 de Febrero de 2017

Al igual que en la mayor parte de los lenguajes de programación, en JavaScript podemos trabajar con arrays (también llamados arreglos, vectores o matrices). Los arrays son de gran importancia ya que permiten organizar series de datos que comparten el mismo nombre pero se diferencian por un índice.

Para inicializar un arreglo, podemos hacerlo de tal manera que esté vacío para despues agregar datos, o poniendo los datos del arreglo desde un inicio:
var nuevoArreglo = new Array();
var nuevoArreglo = ['Dato1', 'Dato2', numero1, numero2];

El acceso a los datos del arreglo se hace por medio de un índice. Cabe mencionar que los datos inician desde la posición 0. Si tenemos el siguiente arreglo:
var materias = ['Cálculo', 'Álgebra', 'Química', 'Física', 'Matemáticas', 'Geometría', 'Inglés'];
El valor de materias[0] sería Cálculo, materias[3] sería Física y materias[6] sería Inglés

Funciones de arreglos

.concat(e1,e2,...,eN)
Devuelve la concatenación de e1, e2, …, eN con el array sobre el que se invoca.
.join('separador')
Une los elementos de un array en una cadena de texto donde cada elemento está separado por ‘separador’.
.push(x)
Añade x al final del array como nuevo (o nuevos) elemento, y devuelve la nueva longitud del array.
.pop(x)
Elimina el último elemento del array y lo devuelve.
.reverse()
Invierte el orden de los elementos en el array (el final pasa a ser el principio).
.sort()
Si no recibe parámetros, ordena los elementos del array por orden alfabético (que no coincide con el numérico), quedando el array modificado.
.shift()
Elimina el primer elemento del array y lo devuelve.
.unshift(x)
Añade x al principio del array como nuevo (o nuevos) elementos.
.splice(ind,f,x)
Modifica el array añadiendo(f=0) o eliminando(f=1) el elemento x en la posición ind (desplazando a los existentes).
.slice(firstIn,lastOut
Devuelve un array con los elementos extraídos entre los índices firstIn y lastOut-1.
.indexOf(x)
Busca x dentro del array y devuelve la posición de la primera ocurrencia.
