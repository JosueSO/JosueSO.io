JavaSript

Fecha: 20 de Febrero de 2017

Es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

JavaSript fue desarrollado originalmente por Brendan Eich de Netscape con el nombre de Mocha en 1995, el cual fue renombrado posteriormente a LiveScript, para finalmente quedar como JavaScript. Este nombre fue dado debido a la reciente compatibilidad con la tecnología Java en el navegador web Netscape Navigator en diciembre del mismo año.
