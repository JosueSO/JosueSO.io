Listas en HTML

Fecha: 1 de Febrero de 2017

Las listas representan uno de los instrumentos más difundidos para organizar la información dentro de los sitios web. Una de sus características principales es la de proporcionar un cuadro claro y sintético del tema tratado.

Listas ordenadas

Este tipo de listas son utilizadas cuando queremos ordenar el contenido de una lista. Un ejemplo de esto es el indice de un libro:

<ol>
  <li>Capítulo 1</li>
  <li>Capítulo 2</li>
  <li>Capítulo 3</li>
</ol>

  1. Capítulo 1
  2. Capítulo 2
  3. Capítulo 3

Listas no ordenadas

Este tipo de listas son utilizadas cuando listamos elementos en el que el orden no importa. Por ejemplo una lista de colores:

<ul>
  <li>Rojo</li>
  <li>Azul</li>
  <li>Verde</li>
</ul>

  • Rojo
  • Azul
  • Verde

Listas de descripción

Este tipo de listas son utilizadas cuando queremos listar elementos y agregar una descripción del mismo. Por ejemplo:

<dl>
  <dt>HTML</dt>
  <dd>Lenguaje para especificar el contenido de una página Web</dd>
  <dt>CSS</dt>
  <dd>Lenguaje para especificar la apariencia de una página Web</dd>
  <dt>JavaScript</dt>
  <dd>Lenguaje para especificar comportamiento</dd>
</ol>

HTML
Lenguaje para especificar el contenido de una página Web
CSS
Lenguaje para especificar la apariencia de una página Web
JavaScript
Lenguaje para especificar comportamiento
