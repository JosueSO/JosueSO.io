HTML

Fecha: 27 de Enero de 2017

HTML (Hypertext Markup Lenguage) es un documento de texto que adquiere la forma de una página Web. Para definir la estructura de la página, HTML hace uso de elentos, los cuales se definen como etiquetas y estan encerradas entre paréntesis angulares < >

Existen dos tipos de etiquetas: de apertura y de cierre. El nombre de la etiqueta de cierre debe ser el mismo que el de apertura pero se distingue por una diagonal. El contenido de un elemento se encierra entre etiquetas:

<etiqueta> contenido </etiqueta>

Las etiquetas pueden tener atributos los cuales se especifican en la etiqueta de apertura. Los atributos contienen Información adicional acerca del contenido.
